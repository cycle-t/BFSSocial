BFSSocial 项目说明
==================

项目作用：
本项目是一个用于微课、课堂演示和答辩展示的 BFS 可视化静态网页。页面围绕“六度空间中的最短关系链”展开，用交互方式讲解广度优先搜索在无权图中的最短路径问题。

根目录内容说明：
- `.github/`：GitHub 仓库配置目录，当前主要用于 GitHub Pages 自动部署。
- `assets/`：页面使用的图片与图标资源目录。
- `index.html`：页面结构文件，定义加载页、原理讲解页、场景演示页、习题页和反馈页。
- `styles.css`：样式文件，负责整体视觉风格、组件样式和响应式布局。
- `script.js`：交互逻辑文件，包含 BFS 可视化、图场景切换、习题评分和“腐烂的橘子”小实验。
- `README.txt`：当前说明文件，用于概述项目用途和根目录文件职责。

运行方式：
直接打开 `index.html` 即可浏览项目；如果需要更接近线上环境，也可以使用任意静态服务器启动。

补充说明：
本项目不依赖前端构建工具，适合作为静态站点直接部署。
