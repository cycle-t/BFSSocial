# BFSSocial

一个面向微课/答辩展示的 BFS 可视化前端项目。它把“六度空间中的最短关系链”做成“独立加载首页 + 4 个教学界面”的流程，既适合课堂讲解，也适合比赛展示。

## 项目定位

本项目围绕广度优先搜索（BFS）在无权图中的最短路径问题展开，核心目标是把抽象算法转成学生容易理解的交互流程：

1. 独立加载页面（首页）
2. BFS 原理讲解界面
3. 场景演示界面
4. 课后习题界面
5. 问卷反馈界面

其中第 3 个界面保留了原本较完整的“场景实验”能力，并进一步融入到完整教学流程中。

## 功能亮点

- 独立首页 + 四界面串联：加载页不并列展示，进入后按 1-4 顺序学习。
- BFS 可视化演示：按层扩散、队列变化、访问顺序、父节点回溯、最短路径锁定全过程可见。
- 多场景切换：内置多个社交关系图模板，便于课堂举例和答辩演示。
- 交互式实验：支持切换起点/终点、单步播放、自动播放、恢复原图。
- 图结构编辑：支持拖动人物调整位置、添加新人物、补边连线，现场演示路径变化。
- 即时小测：演示界面内置预测练习，帮助学生跟随 BFS 队列节奏理解算法。
- 课后习题：单独页面提供自测题和即时评分反馈。
- 问卷反馈：支持展示问卷图片，并预留公众号二维码展示区域。

## 页面结构

### 1. 加载页面

- 用于课程开场和情境导入。
- 展示学习目标、系统加载进度、模块准备状态。
- 作为独立首页，不参与后续 1-4 教学页并列导航。
- 点击进入后直接跳转到 BFS 原理讲解界面。

### 2. BFS 原理讲解界面

- 解释 BFS 的按层搜索机制。
- 强调为什么队列是核心结构。
- 对比 BFS 和 DFS 的差异。
- 用“社交网络扩散”帮助学生理解无权图最短路径。

### 3. 场景演示界面

- 是本项目的核心实验区。
- 左侧为关系网络动画画布，右侧为算法状态解释区。
- 可实时观察：
  - 队列 Queue
  - 访问顺序
  - 层级展开
  - 父节点映射
  - 节点详情
  - 伪代码高亮
  - 最短路径回溯

### 4. 课后习题界面

- 提供独立的课后检测。
- 支持提交答案后即时评分。
- 每题会返回解析，便于课堂复盘或课后自学。

### 5. 问卷反馈界面

- 用于收集学生、教师、评委的使用感受。
- 包含问卷图片展示（`assets/questionnaire.png`）。
- 预留公众号展示区域，可替换成真实二维码与项目传播信息。

## 技术栈

- 原生 JavaScript
- Canvas 2D

项目不依赖任何前端框架，也不需要构建步骤，适合直接部署为静态网站。

## 本地运行

因为是纯静态页面，直接打开 `index.html` 即可运行。

如果你希望更接近线上环境，也可以用任意静态服务器启动项目，例如：

```powershell
cd BFSSocial
python -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 核心文件说明

### `index.html`

- 定义五个界面的页面结构。
- 包含导航、讲解内容、实验区、习题区和反馈区。

### `styles.css`

- 负责整体视觉风格与响应式布局。
- 包含实验区、加载页、习题页、反馈页等全部样式。

### `script.js`

- 实现 BFS 搜索步骤生成与状态更新。
- 驱动画布动画、页面切换、加载进度、习题评分。

## BFS 实验区支持的交互

- 选择不同起点和目标人物
- 自动播放 / 单步播放 / 上一步 / 重播
- 切换关系图场景
- 恢复原始图结构
- 拖动节点调整人物位置
- 单击节点查看详情
- Shift + 单击两个节点建立关系
- 双击空白区域添加新人物
