BFS 为什么优于 DFS

BFS 先把近处找完再向外扩，第一条抵达目标的路径就是最短关系链。

DFS 往往一条路走到底再回头，不适合优先寻找无权图中的最短路。

    核心伪代码

    同步高亮
    1. queue <- [start], seen <- {start}
    2. parent[start] <- null, dist[start] <- 0
    3. while queue is not empty
    4. v <- dequeue(queue)
    5. if v == target: break
    6. for each neighbor u of v
    7. if u not in seen: parent[u]=v; enqueue(u)
    8. reconstruct path(target, parent)
